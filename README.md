# 📚 TechBookHub

**Empowering your tech journey with the best resources.**

TechBookHub ([techbookhub.github.io](https://techbookhub.github.io)) adalah platform edukasi independen yang menyediakan referensi teknologi informasi berkualitas dengan harga terjangkau. Kami hadir untuk membantu pelajar dan profesional IT di Indonesia meningkatkan kompetensi melalui akses ke materi belajar terstruktur.

---

## 🎯 Misi

Kami percaya bahwa peningkatan skill IT seharusnya tidak dibatasi oleh biaya tinggi. TechBookHub menyediakan alternatif terjangkau tanpa mengorbankan kualitas materi pembelajaran, sehingga mempercepat penguasaan skill teknis dan mendukung pertumbuhan karier di bidang teknologi informasi.

---

## 📖 Kategori E-book

Koleksi e-book dikurasi dalam **lima pilar utama**:

| Kategori | Cakupan |
|---|---|
| **Networking & Infrastructure** | Desain jaringan, routing, switching, SD-WAN |
| **Programming & Development** | Pengembangan aplikasi, scripting, otomasi |
| **Security & Assurance** | Keamanan siber, penetration testing, GRC, compliance |
| **Cloud & Virtualization** | Arsitektur cloud, container, virtualisasi, IaC |
| **Data Science & AI** | Machine learning, data analytics, artificial intelligence |

---

## ✨ Fitur Website

- **Katalog Produk Interaktif** — Tampilan grid dengan cover e-book, harga, dan detail lengkap.
- **Filter Kategori** — Filter buku berdasarkan lima kategori utama.
- **Pencarian** — Cari buku berdasarkan judul atau kata kunci.
- **Modal Detail Produk** — Popup informasi lengkap (deskripsi, topik, target audience, sample pages).
- **Share ke Sosial Media** — Bagikan produk ke WhatsApp, Telegram, LinkedIn, Instagram, atau salin URL.
- **Responsive Design** — Dioptimalkan untuk desktop maupun perangkat mobile.
- **Psychological Pricing** — Menampilkan harga coret dan badge diskon untuk transparansi.
- **Image Zoom** — Lihat sample halaman buku secara detail.
- **Load More / Pagination** — Memuat buku secara bertahap untuk performa optimal.
